# Nexus Forge V6: The Ultimate AI Bot Architect

Nexus Forge V6 is a comprehensive bot management and deployment system designed to streamline the creation and deployment of AI-powered bots.

## Features

- **Model Routing**: Intelligently routes tasks between different AI models.
- **Multi-File Suite**: Generates all necessary files for bot deployment.
- **Intelligent Slugify**: Converts repository names to GitHub-compliant slugs.
- **Automatic Dependency Mapper**: Lists all used libraries in `requirements.txt`.
- **GitHub & Deployment Suite**: Automates GitHub configuration, commits, and version control.
- **Personal Assistant Features**: Includes error auto-correction and usage statistics.

## Quick Start

1. Clone the repository.
2. Install dependencies using `pip install -r requirements.txt`.
3. Run the application using `python main.py`.
